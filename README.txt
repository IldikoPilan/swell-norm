# RESOURCES CREATED FOR COLING 2018 SUBMISSION NR. 694

List of the files and folders included:

swe_nst_g2p_2			A grapheme-to-phoneme (g2p) conversion system for Swedish using CMUSphinx
spec_blog_FT_1.bin		The FastText embeddings trained on L2 relevant Swedish coprora and blog texts
phon_sim.py			The Python code for computing phonological similarity
phon_features.csv		The binary phonological features used 
swe_L2_errors_SUBM.csv		The evaluation data with L2 Swedish learner errors and their corrections						

More information about the installation and running of the g2p system:
https://github.com/cmusphinx/g2p-seq2seq 

More information about the FastText embeddings:
https://github.com/facebookresearch/fastText/blob/master/pretrained-vectors.md
